# Productivity Timer 1.0.0

## New

- A per-user Windows installer that needs no administrator access.
- Automatic startup at Windows sign-in, enabled during installation and repaired on launch.
- A real tray tooltip showing the last successful reminder and the next scheduled reminder.
- Temporary pause and quit controls that reset on the next app start or Windows sign-in.
- Clean uninstall support through Windows Settings.

## Notes

This release is unsigned, so Windows SmartScreen may show a warning. Click **More info**, confirm the filename is `ProductivityTimer-Setup.exe`, then click **Run anyway**. Do not disable SmartScreen. Managed computers may require administrator approval.
