# Productivity Timer 1.0.1

## Fixed

- Reminders now appear. Earlier builds sent no notifications and the tray tooltip stayed on `Last: never`, because the app talked to Windows under a notification identity it never registered, so Windows dropped every toast. The app now registers that identity at launch and delivers the reminder.

## Notes

This release is unsigned, so Windows SmartScreen may show a warning. Click **More info**, confirm the filename is `ProductivityTimer-Setup.exe`, then click **Run anyway**. Do not disable SmartScreen. Managed computers may require administrator approval.
